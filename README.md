# MahindraAssignmentApp
This application displays a list of notifications. 
Users can view notification titles, expand notifications to see descriptions,
navigate to Google Maps using a hardcoded location, and view notification tag and timestamps.

# API Details
Base URL : https://jsonplaceholder.typicode.com/

Endpoint : GET /posts

# Tech Stack

1) MVVM with Clean Architecture
2) Retrofit for REST API 
3) StateFlow and Jetpack Compose
4) Hilt Dependency Injection
5) Kotlin Coroutines
